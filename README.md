# novnc

Install `xfce4` into Ubuntu Server:
```bash
sudo apt install xfce4 -y
```

Install tigervnc server:
```bash
sudo apt install tigervnc-standalone-server -y
```

Install novnc:
```bash
sudo snap install novnc
```

Start VNC Server:
```bash
vncserver
```

Start VNC Server:
```bash
novnc --vnc localhost:5901
```
